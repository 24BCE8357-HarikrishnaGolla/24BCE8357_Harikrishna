# Ecommerce Management System

## Overview

A Node.js, Express.js, and MongoDB based Ecommerce Backend Management System.

## Features

- JWT Authentication
- Role Based Authorization
- Product Management
- Category Management
- Cart Management
- Order Management
- Product Dashboard
- Sales Dashboard
- Search, Sorting, Pagination
- MongoDB Aggregation Pipelines

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- JWT
- bcryptjs
